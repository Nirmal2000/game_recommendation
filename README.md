# Game Recommendation

## Installation

### Backend

```bash
cd backend/
pip install -r requirements.txt
```

### Frontend

```bash
cd frontend/
npm i
```

## Running the app

### Backend

```bash
cd backend/
python app.py
```

### Frontend

```bash
cd frontend/
npm start
```
